# EDA and Modeling Branch

This branch contains the exploratory data analysis and multinomial logistic regression model for the Nashville SOPP project.

## Branch Contents

```
EDA-and-modeling/
├── .github/                         ← GitHub Classroom feedback configuration
├── data/model_outputs/              ← Model output files (coefficients, predictions, figures)
├── Baseline_Model_Jupyter.ipynb     ← Baseline MLR model estimation and results
├── eda.ipynb                        ← Exploratory data analysis and figures
├── modeling_updated.ipynb           ← Updated MLR model with interactions and splines
└── README.md                        ← This file
```

## What This Branch Does

### `eda.ipynb`
Performs exploratory data analysis on the cleaned, merged dataset. Generates the following figures:
- **Figure 1** — Distribution of traffic stop outcomes (warning, citation, arrest)
- **Figure 2** — Comparison of wealth index distributions across neighborhood definitions (0.5mi, 1mi, KNN-30)
- **Figure 3** — Spatial map of log-scaled median housing prices across Nashville
- **Figure 4** — Boxplots of wealth index by stop outcome

### `Baseline_Model_Jupyter.ipynb`
Fits an initial baseline Multinomial Logistic Regression model using `statsmodels`. Includes:
- Model specification with wealth index, race, sex, and age
- Initial coefficient estimates and odds ratios
- Preliminary assumption checks

### `modeling_updated.ipynb`
Full model estimation pipeline. Includes:
- VIF analysis (max VIF ≈ 2.67 — no severe multicollinearity)
- Box–Tidwell linearity tests (age flagged as nonlinear → spline used)
- IIA sensitivity analysis
- Final MLR with wealth × race and wealth × sex interaction terms
- Age modeled with natural cubic spline (df=4)
- Cluster-robust standard errors (clustered by year × geographic grid cell)
- Temporal train/test split: 2013–2015 train, 2016 test
- Predicted probability plots by wealth, race, sex, and age (Figures 9–11)

### `data/model_outputs/`
Stores output files generated by the modeling notebooks, including coefficient tables, predicted probability CSVs, and saved figures.

## Required Input Data

This branch reads from the cleaned, merged dataset produced by the `cleaning-and-wrangling` branch. You will need either:
- `df_main_clean.csv` or `df_main_clean.csv.zip` — the merged policing + wealth index dataset

Download or pull this file from the `datasets` or `cleaning-and-wrangling` branch before running notebooks.

## How to Run

```bash
# 1. Switch to this branch
git checkout EDA-and-modeling

# 2. Install dependencies
pip install pandas numpy matplotlib seaborn statsmodels scikit-learn patsy jupyter

# 3. Run in order
jupyter notebook eda.ipynb
jupyter notebook Baseline_Model_Jupyter.ipynb
jupyter notebook modeling_updated.ipynb
```
